Important FAQ

The **DOM (Document Object Model)** is like a map of a webpage that shows all the elements, such as paragraphs, buttons, and images, as a tree structure. It lets me use JavaScript to interact with and change these elements, like updating text, adding styles, or removing items. So, when I change something using JavaScript, the changes immediately reflect on the webpage. This helps make webpages dynamic and more interactive for users.
To select an element from the DOM, you can use `document.getElementById()` if the element has a unique ID, like this: `document.getElementById("myId")`. Another way is using `document.querySelector()`, which lets you select elements with CSS selectors, like this: `document.querySelector(".myClass")` for a class or `document.querySelector("div")` for a tag. These methods let me grab elements from the page so I can manipulate them with JavaScript.
**Event delegation** is a technique in which a single event listener is added to a parent element instead of multiple listeners on individual child elements. This approach takes advantage of event bubbling, where events triggered on a child element propagate up to its parent, allowing the parent to capture and handle the events from any of its children. Event delegation is useful because it improves performance by reducing the number of event listeners, it handles dynamic content effectively by automatically working for new child elements without requiring additional code, and it simplifies the code structure by managing many child elements with just one event listener, making maintenance easier.
To manipulate an element's attributes and styles using the DOM in JavaScript, I can use specific methods and properties. For attributes, I can use `setAttribute`, `getAttribute`, and `removeAttribute`. For example, to set an attribute, I might write `myElement.setAttribute("src", "image.jpg")`, which sets the `src` attribute of the element. To get an attribute value, I can use `myElement.getAttribute("src")`, and to remove it, I'd use `myElement.removeAttribute("src")`. When it comes to styles, I can directly modify the `style` property of an element. For instance, I can change the background color with `myElement.style.backgroundColor = "blue"` or adjust the font size using `myElement.style.fontSize = "20px"`. I can also set multiple styles at once by using `myElement.style.cssText = "color: white; padding: 10px;"`, allowing me to easily update the element's appearance.
